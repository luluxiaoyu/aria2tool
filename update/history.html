# Aria2 Integration Tool V1.0

# V1.0

发布日期：2021年5月1日

  • 支持联网下载Aria2，AriaNG，以及AriaNG
  • 支持Hash校验下载文件完整性，每一次启动都校验Aria2完整性
  • 程序启动时自动启动Aria2主程序
  • 支持Aria2日志输出，屏蔽空白日志
  • 支持程序内对Aria2的开启，关闭，重启
  • 支持循环检测Aria2运行状态
  • 添加帮助文档，更新记录页面
