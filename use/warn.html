# 注意

# 1.请不要手动替换aria2主程序

目前程序每一次启动都会进行联网校验aria2c.exe的SHA1,MD5,CRC32的值是否和程序资源库相符

不符合的会被程序删除！

以后会添加关闭哈希校验的功能吧==

# 2.不要删除download目录

不要删除download目录，即使你修改了下载路径

在这个文件夹里面还有一个data文件夹

这里保存了Aria2的会话数据！
