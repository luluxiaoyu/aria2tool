# 使用方法

# 下载

你可以在下载页面下载到最新版本的程序

后期将加入自动更新功能！

下载后你会得到一个这样的文件

# 使用前的准备

因为程序默认下载文件的地址就在程序根目录下面的download文件夹

所以，请把本软件放在一个空间较多的磁盘中

然后新建一个文件夹，把本软件放在一个空文件夹中（因为运行会生成其它文件在根目录）

注意！

请不要修改软件的名字，哪怕是一个符号！！！

要改的，请创建桌面快捷方式

因为需要修改注册表中IE框架的版本，如果改了名字，修改会失效，导致出现一大堆JS错误！！！

# 开始

双击运行即可！

软件会首先进行补全Aria2资源，这个过程是全自动的

然后主页面就出来了

添加下载链接即可！
